---
title: Swift package plugins
year: 2022
---
A swift script that can take actions on an xcode project or swift package
Can be private, or general purpose

Doesn't bring in any runtime content

Commands plugins
Format source code, lint, prepare for distribution

Build tool plugins
Generate source files, resources. applied to each target

SwiftFormat would be a very useful one, generally.
Many others good for custom actions.

There's another talk for creating plugins
