---
title: Programmatic UI with previews
year: 2023
---

* New `#Preview("Name") {}` macro
* Fast re-compilation
* Use one or more bits of content in a preview macro
  * Swiftui
  * nsview
  * nsviewcontroller
* Previews can have configuration
  * traits: device orientation etc.
* Some cool timeline stuff for widgets. It would be nice UI if this was available views.
