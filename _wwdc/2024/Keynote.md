  * Lock an app, requires Face ID to open that app.
  * Hide an app
  * Share specific contacts with an app (like photos)
  * Connect to specific devices on local network
* Messages
  * Tapbacks, ie reactions, now are in color, any emoji
  * Scheduled Send
  * Text formatting, bold underline italics, etc
  * Text effects, shake, ripple, etc
  * Messages via satellite (on iPhone 14+)
* Mail
  * On device categorization
  * Digest view, all recent emails from a business
* Maps
  * Topo maps, with trail and hiking routes. All national parks can save offline
* Wallet
  * tap to cash
* Journal
  * Log state of mind
  * Stats and insights
* Game mode
* Photos
  * Single view, grid at top, collections at bottom
  * People groups
  * Trips section
  * Customize what things appear in various sections

## Apple Intelligence
* Powerful, Intuitive, Integrated, Personal, Private
* Capabilities
  * Create language
    * Understand personal context
    * Prioritize notifications
    * Writing tools, proofread, summarize, revise
    * Built into system text view
  * Images
    * Can personalize generated images based on photo library
    * Image styles: sketch, animation, illustration
  * Take actions
    * Kinda like Siri already is, but more action types
* Architecture, privacy at the core
  * On device processing
  * Aware of personal data, without sending it out
  * On device semantic index
  * Private cloud compute
    * Use large server based models on apple silicon
    * Sends only data relevant to your task to the servers
    * Independent verifications, via a public log
* Experiences
  * Siri
    * More natural, relevant, and personal
    * New look, glowing light around screen
    * Maintains context for chained requests
    * Type to siri, double tap at the bottom of screen
    * On screen awareness
    * In-app actions, via expansions to App Intents
  * Writing tools
    * Rewrite, different version of what you already have. Inline suggestions.
    * Proofread. Grammar and spelling
    * System wide, including third party apps
    * Mail smart reply, with suggestions, and form style details
    * Email summaries in the list, instead of message
  * Notifications
    * Summaries, new focus
  * Genmoji
    * Describe a custom emoji
    * Can make it look like someone from photos
  * Image Playground
    * On device, by themes
    * Can also add custom descriptions
    * Messages, Freeform, keynote, new Image playground app, and an API
    * Image Wand
      * Turn a sketch from notes into an image
      * Circle a sketch to enter an image playground based on that context
  * Photos
    * Cleanup tool removes extra background items
      * lol, they removed a bagpiper
    * Search for very specific things
    * Search in videos
    * On demand custom photo montages
* ChatGPT support in Siri
  * Asks permission to share with chatGPT first, for each request
  * Free, or connect your account
  * Intend to support other models in the future
* Developer SDK
  * Image playground API
  * Writing tools automatic
  * SiriKit without additional work
  * New App Intents
