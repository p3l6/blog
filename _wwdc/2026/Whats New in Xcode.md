---
title: Whats New in Xcode
year: 2026
---

## Workspace customization
* Some buttons have moved from jump bar to toolbar
* Branch picker moved to bottom bar
* Toolbar is customizable, add/remove/move items
* Xcode has new themes, that affect the whole window instead of just code editor
  * Included themes have sliders to adjust intensities
* Can choose different themes per-workspace
* Predictive code issues are subtle, to distinguish from issues in an actual build

## Projects and files
* Template picker for new projects is streamlined
  * No initialization questions
  * Can try out creating projects without saving
* Playgrounds and previews can show for single files outside of projects

## Intelligence
* Agent transcript is now in the editor pane instead of sidebar
* `/plan` command
* Agent chat and file changes appear side-by-side, like a preview canvas
* Sidebar now shows existing and past conversations

## Running apps
* Device hub replaces simulators, has more controls for device settings
* Iphone mirroring apps can be resized, can test this in device hub
* Hub also can work with and show screen for real devices.

## Refining apps
* Agents can do app translations
* Testflight users can provide translation feedback
* Organizer has new insights, overview page, storage metrics, metric goals
* Kick off agent diagnostics from organizer
* New 'top functions' view in instruments, to find some performance problems
